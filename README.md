# NextStep

AI Programming Tutor frontend prototype for adaptive Python practice.

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

The app is implemented with React, TypeScript, TailwindCSS, shadcn-style local UI components, Framer Motion, Recharts, and Monaco Editor.

## Docs

- `docs/FRONTEND_DESIGN.md`: page structure and component breakdown.
- `docs/API.md`: FastAPI contract expected by the frontend.
