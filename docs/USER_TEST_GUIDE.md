## Suggested Student Test

### 1. Register and Sign In

1. Select **Register** on the login page.
2. Create a new student account with test-only information.
3. Sign in using the account you created.
4. Check that the student dashboard opens successfully.

### 2. Complete the Diagnostic Assessment

1. Start the diagnostic assessment when prompted.
2. Answer all questions without looking up answers.
3. Submit the assessment and review the resulting learning path or recommended exercise.
4. Consider whether the initial recommendation matches your confidence level in Python.

### 3. Complete a Practice Exercise

1. Open the recommended exercise.
2. Read the task description, starter code, and test requirements.
3. Write a Python solution in the editor.
4. Select **Run Code** to check the code while working.
5. Select **Submit** when you are ready to save the attempt.
6. Review the result, test feedback, mastery update, and next exercise recommendation.

### 4. Use Learning Support

1. Request hints one level at a time when you are stuck.
2. Ask the AI assistant a question about the current Python topic or your learning progress.
3. Check whether the explanation is understandable and helps you decide the next step.
4. Continue with another exercise if a recommendation is available.

### 5. Review Progress

1. Return to the student dashboard after one or more submissions.
2. Review mastery by knowledge component, learning advice, and the recommended next exercise.
3. Log out and sign in again.
4. Check that your submitted attempts and learning progress are still available.

## Suggested Teacher Test

1. Log in with the teacher account.
2. Review the class heatmap, low-mastery list, weak knowledge components, and recent submissions.
3. Use the student search box to search by name or student ID.
4. Open a student's detail panel from the search result, heatmap, or low-mastery list.
5. Confirm that mastery, weak knowledge components, and recent activity are understandable.
